# All-Sorts
Сортировка массива несколькими методами на выбор. (Консольное приложение).

Консольное приложение. На выбор можно написать массив самому или воспользоваться сгенерированным рандомным массивом нужной вам длинны.
Далее нужно выбрать метод сортировки массива, так же имеется вариант выбрать все.
В результате выводится отсортированный массив и время затраченное на сортировку данного массива для каждого варианта.
